# frontend — jeval UI

React + TypeScript + Tailwind UI платформы оценки должностей. Тёмная/светлая темы,
glass-эффект, единственный акцент — красный `#ff3d00`. Без Material/Bootstrap/UI-китов.

## Запуск

```bash
cd frontend
npm install
cp .env.example .env          # VITE_API_URL → адрес FastAPI-бэкенда
npm run dev                   # http://localhost:5173
npm run build                 # production-сборка в dist/
npm run lint                  # tsc --noEmit
```

## Экраны

| Маршрут | Экран | Назначение |
|---------|-------|-----------|
| `/` | Дашборд | KPI-карты, фильтры, таблица должностей (из API) |
| `/new` | Форма должности | 7-шаговый stepper + живая панель Gate 0, сохранение и запуск оценки |
| `/positions/:id` | Карточка оценки | факторная таблица, формула, QC-флаги, обоснование, переоценка, печать |
| `/compare` | Сравнение | текущая роль + якоря (селекторы), автоматический анализ различий |

Все экраны работают на реальном API (`VITE_API_URL`). Для работы нужен запущенный
бэкенд: `cd backend && uvicorn jeval.api.main:app` (без `ANTHROPIC_API_KEY` включите
в `backend/.env` офлайн-режим `JEVAL_FAKE_AGENT=1`).

## Стек и решения

- **Vite + React 18 + TypeScript** (strict).
- **Tailwind** с токенами через CSS-переменные (`src/index.css`) → темы переключаются
  классом `dark` на `<html>` ([src/lib/theme.tsx](src/lib/theme.tsx)).
- **framer-motion** — только для разворачивания строк и модалок (минимум анимаций).
- Клиент API — [src/lib/api.ts](src/lib/api.ts) (типы в [src/lib/types.ts](src/lib/types.ts)
  синхронны с `jeval/domain/models.py`); маппинг в view-модели —
  [src/lib/mapping.ts](src/lib/mapping.ts); загрузка — хук [src/lib/useFetch.ts](src/lib/useFetch.ts).
- Карточка оценки отражает статусы конвейера: «нет оценки» (Gate 0-превью),
  `cannot_evaluate` (список недостающих блоков), `needs_clarification` / `ready`
  (полная факторная карточка).

## Дизайн-принципы (из брифа)

Дот-статусы вместо цветных бейджей, без иконок перед каждым текстом, без спиннеров
(скелетоны), тени только на модалках, ховер меняет цвет/границу без смещения,
один акцентный цвет. Полный бриф — в [FIGMA_PROMPT.md](FIGMA_PROMPT.md).
